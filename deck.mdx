# Rules of Hooks

1. Only call hooks from React functions

---

Right:
```js
const Component = () => {
  useEffect(() => {
    // Do something...
  })

  return <h1>Yes!</h1>
}
```

---

Wrong:
```js
const wrongFunc = () => {
  useEffect(() => {
    // Do something...
  })

  return 'No';
}
```

---

# Rules of Hooks

2. Always use hooks at the top level of the component

---

Right:
```js
const Component = () => {
  useEffect(() => {
    // Do something...
  })

  return <h1>Yes!</h1>
}
```

---

Wrong:
```js
const Component = ({isWrong}) => {
  if (isWrong) {
    useEffect(() => {
      // Do something...
    })
  }

  return <h1>No!</h1>
}
```

---

# Rules of Hooks

3. Include all useEffect dependencies

---

Right:
```js
useEffect(() => {
  fetchData(id).then(() => {
    ...    
  }).catch(err => {
    ...
  })
}, [id]);
```

---

Wrong:
```js
useEffect(() => {
  fetchData(id).then(() => {
    ...    
  }).catch(err => {
    ...
  })
}, []); // <-- Missing `id`!
```
